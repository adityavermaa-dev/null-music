# Code of Conduct

We want **Null** to be a welcoming project for everyone.

## Our standards

- Be respectful and constructive.
- Assume good intent; ask clarifying questions.
- No harassment, hate speech, or discriminatory language.
- Keep feedback focused on the work, not the person.

## Scope

This applies to all project spaces: issues, pull requests, discussions, and any community channels linked from this repo.

## Enforcement

Project maintainers may warn, edit/remove content, or temporarily/permanently restrict participation for behavior that violates these standards.

## Reporting

If something happens that makes you (or others) feel unsafe or unwelcome, open an issue with minimal details (avoid posting sensitive info), and a maintainer will follow up.
