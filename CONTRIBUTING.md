# Contributing

Thanks for your interest in contributing to **Null**.

## Quick start

- Fork the repo and create a feature branch.
- Install deps: `npm install`
- Run dev stack: `npm run dev:all`
  - Web UI: Vite dev server
  - API: `server.mjs`
- Run checks before PR:
  - `npm run lint`
  - `npm test`

## What to include in PRs

- A clear description of the change and the motivation.
- Screenshots for UI changes.
- Notes about any new env vars.

## Reporting bugs

Please include:
- Steps to reproduce
- Expected vs actual behavior
- Browser/device info
- Relevant server logs (redact tokens/keys)

## Security issues

If you believe you found a security issue, please follow SECURITY.md.
